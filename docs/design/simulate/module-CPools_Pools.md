# CPools_Pools 模块文档

## 1. 模块概述

`CPools_Pools` 是量化交易模拟系统的股票池管理模块，负责加载和管理股票池数据。它支持从CSV文件或数据库加载股票池，并提供了根据市场风格自适应选择股票池的功能。该模块是策略执行的基础，为交易系统提供选股标的。

## 2. 主要功能

- **股票池加载**：从CSV文件或数据库加载股票池数据
- **风格自适应**：根据市场风格自动选择适合的股票池
- **股票池排序**：支持对股票池进行排序
- **数据管理**：管理股票池数据的存储和读取

## 4. 股票池类型

### 4.1 文件类型股票池

- **加载方式**：从CSV文件加载
- **文件路径**：`GLOBALMAP.POOLSPATH + self.stra_code + '.csv'`
- **数据格式**：包含 'code' 和 'date' 列的CSV文件

### 4.2 数据库类型股票池

- **加载方式**：从数据库加载
- **数据库集合**：`client.strategy_pools`
- **数据格式**：包含 'code'（策略代码）、'date'（日期）和 'pools'（股票列表）字段的文档

## 5. 风格自适应

### 5.1 自动风格适应（'auto'）

该模式会根据大盘、小盘和情绪指数的状态自动选择适合的股票池：

1. **计算指数风格相关性**：
   - 大盘指数（880007）：活筹市值
   - 小盘指数（880823）：微盘股
   - 情绪指数（880785）：市场情绪

2. **选择逻辑**：
   - 如果情绪指数值最低且小于25，选择 'strdays_all' 股票池
   - 否则，如果小盘指数值最低且小于25，选择 'strdays_all' 股票池
   - 否则，选择 'strong' 股票池

### 5.2 收益率自适应（'autorate'）

该模式会比较不同股票池的历史收益率，选择表现较好的股票池：

1. **计算收益率**：
   - 计算 'strong' 股票池的平均收益率
   - 计算 'strdays_all' 股票池的平均收益率

2. **选择逻辑**：
   - 如果 'strong' 股票池的收益率大于等于 'strdays_all' 股票池，选择 'strong' 股票池
   - 否则，选择 'strdays_all' 股票池

## 7. 使用示例

### 7.1 初始化股票池

```python
from FQMarket.Simulate.CPools_Pools import CPoolsData

# 初始化股票池（从CSV文件加载）
pools = CPoolsData(
    code='strong',
    pools_stra=['files', ['default', False], 0, None]
)

# 初始化股票池（从数据库加载，自适应风格）
pools = CPoolsData(
    code='strong',
    pools_stra=['DB', ['default', False], 0, 'auto']
)
```

### 7.2 加载股票池数据

```python
# 加载2023-01-01的股票池数据
stock_list = pools.loadPools(end_date='2023-01-01')
print(stock_list)
```

### 7.3 保存股票池数据

```python
# 保存股票池数据到CSV文件
pools.savePools2CSV()
```

## 8. 注意事项

1. **除权除息处理**：当前代码未处理除权除息，可能会影响股票池的选股结果
2. **数据依赖**：依赖于数据库中的策略池数据，需要确保数据更新及时
3. **风格判断**：风格自适应的判断逻辑可能需要根据市场环境进行调整
4. **性能考虑**：对于大规模股票池，可能存在性能瓶颈

## 9. 代码优化建议

1. **除权除息处理**：实现除权除息调整逻辑，确保股票池数据的准确性
2. **缓存机制**：添加缓存机制，减少数据库查询次数，提高性能
3. **参数优化**：将风格判断的参数外置，便于调整
4. **多源数据**：支持从多个数据源加载股票池数据
5. **错误处理**：添加错误处理机制，提高系统稳定性
