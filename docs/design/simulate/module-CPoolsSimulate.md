# CPoolsSimulate 模块文档

## 1. 模块概述

`CPoolsSimulate` 是量化交易模拟系统的核心类，继承自 `CPoolsBase`，负责协调整个交易流程，包括开盘前检查、收盘前检查、净值计算等核心功能。它是连接各个策略模块的中心，实现了完整的模拟交易流程。

## 2. 主要功能

- **开盘前检查**：检查新开仓情况及运算盘中止盈价格
- **收盘前检查**：检查仓位收益情况，给出止损标的及到期标的
- **净值计算**：计算每日净值，管理交易流程
- **策略协调**：协调买入、卖出、空仓等策略的执行
- **交易执行**：执行买入、卖出操作，管理仓位

### 3.3 主要方法

#### 3.3.1 开盘前检查

**DoWorks0800**
- **功能**：开盘前检查新开仓情况及运算盘中止盈价格
- **参数**：
  - `end_date`：结束日期（可选）
  - `title`：通知标题（默认："菜篮子增强"）
  - `client`：数据库客户端（默认：DATABASE）
- **返回值**：无
- **说明**：该方法会在开盘前执行，检查今日是否开仓，计算持仓的止盈价格，并发送通知

#### 3.3.2 收盘前检查

**DoWorks1500**
- **功能**：收盘前检查仓位收益情况，给出止损标的及到期标的
- **参数**：
  - `end_date`：结束日期（可选）
  - `title`：通知标题（默认："菜篮子增强"）
  - `client`：数据库客户端（默认：DATABASE）
- **返回值**：无
- **说明**：该方法会在收盘前执行，检查持仓是否需要止损或到期平仓，并发送通知

#### 3.3.3 净值计算

**calunetvalue**
- **功能**：计算每日净值，执行交易流程
- **参数**：
  - `end_date`：结束日期（可选）
  - `start_date`：开始日期（可选）
  - `renew`：是否重新计算（默认：False）
  - `clean`：是否清仓（默认：False）
  - `client`：数据库客户端（默认：DATABASE）
- **返回值**：无
- **说明**：该方法是核心方法，执行完整的交易流程，包括：
  - 加载历史净值
  - 检查市场状态
  - 执行开盘卖出操作
  - 执行买入操作
  - 计算盘中最大仓位和最大净值
  - 执行收盘卖出操作
  - 更新仓位和净值
  - 保存交易记录

## 4. 交易流程

### 4.1 完整交易流程

1. **初始化**：设置策略参数、资金规模、仓位数量等
2. **数据加载**：加载股票池数据和市场数据
3. **市场状态检查**：判断是否满足开仓条件
4. **开盘操作**：
   - 加载昨日持仓
   - 执行开盘卖出（止盈）
   - 执行买入操作
5. **盘中监控**：计算盘中最大仓位和最大净值
6. **收盘操作**：
   - 执行收盘卖出（止损、到期平仓）
   - 更新仓位和持仓天数
7. **净值计算**：计算当日净值和成功率
8. **数据保存**：保存交易记录和净值数据

### 4.2 策略执行顺序

1. **空仓策略**：判断是否开仓
2. **买入策略**：执行买入操作
3. **卖出策略**：执行卖出操作（开盘止盈、收盘止损、到期平仓）
4. **仓位管理**：调整仓位规模

## 6. 使用示例

### 6.1 初始化模拟交易

```python
from FQMarket.Simulate.CPoolsSimulate import CPoolsSimulate

# 初始化模拟交易
# code参数：[策略代码, 回测代码1, 回测代码2, ...]
sim = CPoolsSimulate(
    code=['strong', 'backtest_001'],
    position_stra=['default_funds'],
    buy_stra=['buy_stra_base', [['open', 0, 5]]],
    sell_stra=['sell_stra_base', [0.05], [-0.06], [5]],
    empty_stra=None,
    clean_stra=['clean_no_code', None],
    pools_stra=None
)
```

### 6.2 执行净值计算

```python
# 计算净值，从2023-01-01到2023-12-31
sim.calunetvalue(
    start_date='2023-01-01',
    end_date='2023-12-31',
    renew=True,  # 重新计算
    clean=False  # 不清仓
)
```

### 6.3 执行开盘前检查

```python
# 执行开盘前检查
sim.DoWorks0800(
    end_date='2023-01-01',
    title='我的策略'
)
```

### 6.4 执行收盘前检查

```python
# 执行收盘前检查
sim.DoWorks1500(
    end_date='2023-01-01',
    title='我的策略'
)
```

## 7. 注意事项

1. **除权除息处理**：当前代码未处理除权除息，可能会影响持仓成本和收益率计算的准确性
2. **数据库依赖**：系统依赖MongoDB存储数据，需要确保数据库连接正常
3. **参数调优**：策略参数需要根据市场环境进行调优
4. **回测性能**：对于大规模回测，可能存在性能瓶颈
5. **风险控制**：需要根据实际情况调整风险控制参数

## 8. 代码优化建议

1. **除权除息处理**：实现除权除息调整逻辑，包括价格和持仓数量的调整
2. **并行计算**：对于大规模回测，考虑使用并行计算提高性能
3. **参数优化**：实现参数自动优化功能，根据历史数据自动调整策略参数
4. **风险控制**：增强风险控制机制，如最大回撤控制、波动率控制等
5. **策略组合**：支持多种策略的组合使用，提高策略的适应性
