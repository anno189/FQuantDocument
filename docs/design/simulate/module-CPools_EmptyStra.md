# CPools_EmptyStra 模块文档

## 1. 模块概述

`CPools_EmptyStra` 是量化交易模拟系统的空仓策略模块，负责判断是否应该空仓。它支持多种空仓策略，包括基于净值、指数、成交量等因素的判断。该模块是交易系统的重要组成部分，决定了何时应该停止开仓，以控制风险。

## 2. 主要功能

- **空仓策略执行**：根据配置的空仓策略执行空仓判断
- **多策略支持**：支持多种空仓策略的组合使用
- **市场状态判断**：基于市场状态判断是否应该空仓
- **净值判断**：基于净值变化判断是否应该空仓
- **成交量判断**：基于成交量判断是否应该空仓

### 3.4 主要方法

#### 3.4.1 获取策略值

**getStraValues**
- **功能**：执行空仓策略，判断是否应该空仓
- **参数**：
  - `end_date`：结束日期（可选）
  - `stra_networth`：策略净值数据（默认：空DataFrame）
- **返回值**：是否空仓（True/False）、空仓原因

### 3.5 具体空仓策略

#### 3.5.1 不空仓策略

**empty_no_code**
- **功能**：默认不空仓策略
- **参数**：
  - `end_date`：结束日期（可选）
  - `stra_networth`：策略净值数据（默认：空DataFrame）
  - `params`：参数（可选）
- **返回值**：False（不空仓）、None

#### 3.5.2 基本空仓策略

**empty_stra_base**
- **功能**：基于净值变化的空仓策略
- **参数**：同 `empty_no_code` 方法
- **返回值**：是否空仓（True/False）、空仓原因
- **说明**：当日跌幅达到某个值时，根据指数均线位置判断是否空仓

#### 3.5.3 指数空仓策略

**empty_stra_index3days**
- **功能**：基于指数状态的空仓策略
- **参数**：同 `empty_no_code` 方法
- **返回值**：是否空仓（True/False）、空仓原因
- **说明**：当指数处于下跌趋势时，判断是否空仓

#### 3.5.4 成交量空仓策略

**empty_amount_shizhi**
- **功能**：基于成交量的空仓策略
- **参数**：同 `empty_no_code` 方法
- **返回值**：是否空仓（True/False）、空仓原因
- **说明**：当成交量枯竭时，判断是否空仓

#### 3.5.5 涨跌停比例空仓策略

**empty_updown_proportion**
- **功能**：基于涨跌停比例的空仓策略
- **参数**：同 `empty_no_code` 方法
- **返回值**：是否空仓（True/False）、空仓原因
- **说明**：当连续跌停股过多时，判断是否空仓

#### 3.5.6 股票池数量空仓策略

**empty_strpools_count**

- **返回值**：是否空仓（True/False）、空仓原因
- **说明**：当多个策略的选股结果显示高风险时，判断是否空仓

#### 3.5.7 策略负反馈空仓策略

**empty_stra_str_negative**
- **功能**：基于策略负反馈的空仓策略

**empty_stra_downtimes**
- **功能**：基于连续下跌的空仓策略

#### 3.5.9 无新高空仓策略

**empty_stra_nohhv**
- **功能**：基于无新高的空仓策略
- **参数**：同 `empty_no_code` 方法
- **返回值**：是否空仓（True/False）、空仓原因
- **说明**：当近N日没有新高时，判断是否空仓

## 4. 空仓策略流程

### 4.1 基本空仓流程

1. **遍历空仓策略**：遍历配置的所有空仓策略
2. **执行空仓判断**：对每个策略执行空仓判断
3. **组合判断结果**：只要有一个策略判断为空仓，则整体判断为空仓
4. **返回结果**：返回是否空仓及空仓原因

### 4.2 常用空仓策略逻辑

- **基本空仓策略**：当日跌幅达到某个值，且指数处于均线下方时空仓
- **指数空仓策略**：指数处于下跌趋势时空仓
- **成交量空仓策略**：成交量枯竭时空仓
- **涨跌停比例空仓策略**：连续跌停股过多时空仓
- **股票池数量空仓策略**：多个策略的选股结果显示高风险时空仓
- **策略负反馈空仓策略**：股票池出现负反馈时空仓
- **连续下跌空仓策略**：连续下跌达到一定程度时空仓
- **无新高空仓策略**：近N日没有新高时空仓

## 5. 空仓条件

### 5.1 基本空仓条件

- **净值下跌**：当日净值下跌超过阈值
- **指数下跌**：指数处于下跌趋势
- **成交量枯竭**：成交量低于阈值
- **涨跌停比例失衡**：连续跌停股过多
- **股票池风险**：多个策略的选股结果显示高风险
- **策略负反馈**：股票池出现负反馈
- **连续下跌**：连续下跌达到一定程度
- **无新高**：近N日没有新高

## 6. 使用示例

### 6.1 初始化空仓策略

```python
from FQMarket.Simulate.CPools_EmptyStra import CPools_EmptyStra

# 初始化空仓策略（默认不空仓）
empty_stra = CPools_EmptyStra(
    params=[['empty_no_code']]
)

# 初始化空仓策略（基本空仓策略）
empty_stra = CPools_EmptyStra(
    params=[['empty_stra_base', -0.0159]]
)

# 初始化空仓策略（组合多个策略）
empty_stra = CPools_EmptyStra(
    params=[
        ['empty_stra_base', -0.0159],
        ['empty_amount_shizhi'],
        ['empty_updown_proportion', 3]
    ]
)
```

### 6.2 执行空仓判断

```python
# 执行空仓判断
empty, message = empty_stra.getStraValues(
    end_date='2023-01-01',
    stra_networth=networth  # 策略净值数据
)

print(f"是否空仓：{empty}")
print(f"空仓原因：{message}")
```
