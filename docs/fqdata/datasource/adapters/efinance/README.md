# EFinance 适配器

EFinance 数据适配器模块，封装 EFinance 金融数据接口。

## 导入

```python
from FQData.DataSource.adapters.efinance import EFinanceAdapter
```

## EFinanceAdapter

EFinance 数据适配器，继承自 `DataSourceAdapter`。

```python
from FQData.DataSource.adapters.efinance import EFinanceAdapter

adapter = EFinanceAdapter()
```
