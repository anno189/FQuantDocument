## 术语

### 无头浏览器

**定义：** 没有可视化界面的浏览器，通过程序控制

**示例：** Selenium 控制的无头 Chrome

### WebDriver

**定义：** Selenium 用于控制浏览器的接口

### CSS 选择器

**定义：** 用于定位 HTML 元素的语法
