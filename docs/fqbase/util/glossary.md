# Util - 术语表

## 阅读路径

🟢 **新手入门**：README → glossary

## 术语

### Converter

数据转换器，将一种类型转换为另一种。

### Transformer

格式转换器，在不同格式间转换（如 dict ↔ DataFrame）。

### ParallelProcess

多进程并行处理封装，适合 CPU 密集型任务。

### ParallelThread

多线程并行处理封装，适合 I/O 密集型任务。

### Safe Divide

安全除法，避免除零错误。

### MD5/SHA256

文件哈希算法，用于文件完整性校验。

### Web Ping

网络连通性检测工具。
