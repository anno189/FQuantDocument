## 组件

### Core（核心配置）

**用途：** 提供基础配置功能

**职责：**
- 环境变量管理（load_env, get_env, reload_env）
- 数据库配置（SETTING, DATABASE）
- 缓存配置（CacheConfig, CacheType）
- 配置监听（ConfigWatcher）

### Business（业务配置）

**用途：** 提供量化交易业务相关配置

**职责：**
- 交易常量（ORDER_DIRECTION, EXCHANGE_ID 等）
- 数据源配置（DataSourceConfig）
- 财务指标映射（FINANCIAL_INDICATORS）
- IP 列表配置

## 数据流

1. **环境变量加载**
   ```
   .env 文件 → load_env() → EnvManager → 内存缓存
   ```

2. **配置读取**
   ```
   get_env() → EnvManager → 返回配置值
   ```

3. **配置监听**
   ```
   配置文件变化 → ConfigWatcher → 回调通知 → 应用更新
   ```

## 依赖

| 依赖 | 版本 | 用途 |
|------|------|------|
| python-dotenv | >=0.19.0 | 加载 .env 文件 |
| fquant-foundation | >=1.0 | 单例模式实现 |

## 设计原则

1. **单一职责**：核心配置和业务配置分离
2. **依赖倒置**：业务配置依赖核心配置
3. **最小暴露**：只导出必要接口
4. **向后兼容**：保持公共 API 稳定
