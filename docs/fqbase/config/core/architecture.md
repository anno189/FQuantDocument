## 概述

Core 核心配置模块的技术架构设计。

## 架构图

```
┌─────────────────────────────────────┐
│         Core 核心配置                 │
├─────────────────────────────────────┤
│  ┌──────────┐  ┌──────────┐       │
│  │ EnvManager│  │  Cache   │       │
│  │ 环境变量  │  │  缓存配置 │       │
│  └──────────┘  └──────────┘       │
│  ┌──────────┐  ┌──────────┐       │
│  │ Database │  │ Config   │       │

│  └──────────┘  └──────────┘       │
└─────────────────────────────────────┘
```

## 组件

### EnvManager
- 负责环境变量加载和读取
- 使用单例模式

### 缓存配置
- 管理多种缓存类型
- 提供统一接口

### 数据库配置
- MongoDB 连接配置
- 路径配置
