# Foundation - 术语表

## 阅读路径

🟢 **新手入门**：README → glossary

## 术语

### Dotty

字典包装器，支持点号深度访问嵌套字典。

### EventBus

事件总线，实现发布-订阅模式的核心组件。

### Event

事件对象，包含事件类型、数据、时间戳。

### Observer Pattern

观察者模式，一种行为设计模式，定义对象间的一对多依赖关系。

### Lifecycle Protocol

生命周期协议，定义服务的初始化、健康检查、关闭等行为接口。

### HealthCheckable

健康检查协议，实现此协议的类可提供健康状态检查。

### Initializable

初始化协议，支持显式初始化。

### Shutdownable

关闭协议，支持优雅关闭。

### ServiceStatus

服务状态枚举，包含服务的各种状态。

### NotificationManager

统一通知管理器，支持多种通知渠道。

### NotificationTemplate

通知消息模板，支持动态参数渲染。
